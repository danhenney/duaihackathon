# 제출용 프로젝트 설명

## 5. 프로젝트 상세 설명

- Why: 투자 인플루언서의 말은 빠르게 퍼지지만 실제 성과는 잘 추적되지 않아, 공개 발언의 책임과 재미를 동시에 보여주는 서비스가 필요했습니다.
- What: 사라했제는 인플루언서, 연예인, 정치인 등이 특정 종목이나 코인에 긍정 의견을 낸 시점부터 현재까지의 수익률을 출처와 함께 보여주는 투자 의견 트래킹 앱입니다.
- How: X, 유튜브, 기사, 텔레그램 등 공개 출처 기반 데이터를 시드로 구축하고, Codex로 프론트엔드·차트·검색·DB 연동을 빠르게 구현했습니다.
- If: 더 많은 공개 의견이 쌓이면 누가 진짜로 먼저 맞혔는지 확인하고 공유하는 새로운 투자 엔터테인먼트 데이터베이스로 확장할 수 있습니다.
